# Nova Recipe Builder — GitHub Pages Free PWA Package

This package is configured for this GitHub repository name:

nova-recipe-builder

## Deploy for free with GitHub Pages
1. Upload/replace these files at the root of your GitHub repository:
   - index.html
   - manifest.json
   - service-worker.js
   - icon-192x192.png
   - icon-512x512.png
2. In GitHub, go to Settings > Pages.
3. Under Build and deployment, set Source to Deploy from a branch.
4. Branch: main
5. Folder: /root
6. Click Save.
7. Open the Pages URL shown by GitHub.

Expected URL:
https://brentlyj-creator.github.io/nova-recipe-builder/

## Install
Open the Pages URL in Microsoft Edge, then use Edge menu > Apps > Install this site as an app.

## Backup
Inside the installed app, use Choose Backup Folder and select a OneDrive synced folder. Then use Backup JSON.
